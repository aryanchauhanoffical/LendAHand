# LendAHand(Currently for Android Users)

LendAHand is an Android application designed to connect individuals in need with job opportunities, NGOs, and support services. The app aims to provide a dignified alternative to begging by offering real-time job listings, support services, and community assistance.

## Features

### For Job Seekers
- 🔍 Search for job opportunities near you
- 📱 View detailed job listings with requirements and pay
- 🗺️ Interactive map view of available jobs
- 📋 Filter jobs by type, location, and pay
- 📞 Direct contact with employers
- 🏥 Find nearby hospitals and medical services
- 🤝 Connect with NGOs for support

### For Employers
- 💼 Post job opportunities
- 📝 Create detailed job listings
- 📍 Set job location and requirements
- 👥 Manage applications
- 📊 Track job posting performance

### For NGOs
- 🏢 Create NGO profiles
- 📢 Post support services
- 📞 Manage contact information
- 📍 Set service locations
- 📊 Track service impact

## Technical Features

- Modern Android UI with Jetpack Compose
- Material Design 3 components
- Interactive Google Maps integration
- Real-time location services
- Offline support
- Secure authentication
- Responsive design for all screen sizes

## Getting Started

### Prerequisites
- Android Studio Arctic Fox or newer
- Minimum SDK: 24 (Android 7.0)
- Target SDK: 34 (Android 14)
